Demos

SVG illustrations can be animated by grouping graphics as desired and giving those groups names to specify their transition type, duration, and delay. The animations in this page are set to auto-play when they scroll into view using Porthole.

Animation Markup How-To

SVG markup must be included inline in the HTML, wrapped in a <div class="animate-svg"></div> element for transtiions to work.


	<div class="animate-svg">
	  <svg>
	    <g id="fade">...</g>
	  </svg>
	</div>

Configuring Play Timing

To have the animation play immediately when the page loads, no matter where it is in the page, give the `div` wrapper the attribute data-animatesvg-init.

<div class="animate-svg" data-animatesvg-init> <svg> <g id="fade">...</g> </svg> </div>

To have the animation play once when it scrolls into the viewport, give the div wrapper the attribute data-scroll-activate.

<div class="animate-svg" data-animatesvg-init> <svg> <g id="fade">...</g> </svg> </div>

To have the animation play every time when it scrolls into the viewport, give the div wrapper the attribute data-scroll-activate and the attribute data-scroll-deactivate.

<div class="animate-svg" data-scroll-activate data-scroll-deactivate> <svg> <g id="fade">...</g> </svg> </div>

Setting Duration

You can set a custom duration on any g element to change it from the default of 800ms. For example, the following example has a g with an ID of rotate-right duration-5000, to make its animation run for 5 seconds.

Custom Delays

You can set a custom delay on any g element to change it from the default, which is just after the prior g finishes its transition. If you set a custom delay, note that it represents the time from the start of the entire animation, so delays can be used to change the order that your groups animate in. For example, the following example has a g with an ID of rotate-right delay-2000, to make its transition delay for 2 seconds before starting.

Building Animations in Adobe Illustrator

If you're editing an SVG in Adobe Illustrator, you can group graphics in the Layers Palette and name their layer as if you are setting the group's ID attribute (with the transition name, duration, and delay.

Tips and Caveats

  • In order to use the layer naming technique above, you'll need at least 2 layers in the pallete. Unnamed layers will go ignored by the Animation script.
  • If more than one layer has the same name, Illustrator will add underscores and other characters to the SVG markup to make its ID unique. It'll still appear the same way in Illustrator however, and the animation engine knows to look for these characters and ignore them.
