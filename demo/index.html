LayerSnap Demos

LayerSnap is a script that builds on top of Snap SVG to enable simple declarative SVG builds by configuring the SVG file itselfâ€“no need to script anything. You can even build animations from the Layers pallete in Adobe Illustrator!

An SVG illustration can be animated by grouping graphics as desired and giving those groups IDs to specify their transition type, duration, and delay. The animations in this page are set to auto-play when they scroll into view (using Porthole, which is explained later in this page).

How-To

Include SnapSVG, layersnap.js, and the layersnap.css in your page.

Next, add some SVG markup inline in the body of your HTML, wrapped in a <div class="layersnap"></div>. To animate a portion of your svg markup, group it in a g element (the g element must be a direct child of the svg element) and configure its animation via that g element's ID value. For example, id="fade" will cause that g element to fade in. 


<div class="layersnap" id="mysvg">
  <svg>
    <g id="fade">...</g>
  </svg>
</div>

To animate the svg, call the Layersnap() function, passing a reference to your svg's wrapper div:


<script>
	var mysvg = document.getElementById( "mysvg" );
	new window.Layersnap( mysvg ).init();
</script>

That's it! From there, the layersnap script will play all g element animations within the svg based on their ID attribute configurations. By default, g elements will animate in source order one after the other, but custom delays and durations allow you to configure the order of elements for richer animations.

Advanced: To create new custom named transitions, you can extend the Layersnap object's transitions property. Here's the drift right transition for example:


<script>
w.Layersnap.prototype.transitions[ "drift-right" ] = function( el, duration ){
	el.attr( {transform: "translate(-30,0)"} );
	el.animate({ opacity: 1, transform: "translate(0,0)" }, duration, mina.easeOut );
};
</script>

Setting Duration

You can set a custom duration on any g element to change it from the default of 800ms. For example, the following example has a g with an ID of rotate-right duration-5000, to make its animation run for 5 seconds.

Custom Delays

You can set a custom delay on any g element to change it from the default, which is just after the prior g finishes its transition. If you set a custom delay, note that it represents the time from the start of the entire animation, so delays can be used to change the order that your groups animate in. For example, the following example has a g with an ID of rotate-right delay-2000, to make its transition delay for 2 seconds before starting.

Replay Buttons

If you want a button that'll replay the animation on click, give the div wrapper the attribute data-layersnap-replay. Every demo in this page except for the interactive ones has a replay button overlayed on it, with opacity: 0.


	<div class="layersnap" data-layersnap-replay>
	  <svg>
	    <g id="fade">...</g>
	  </svg>
	</div>

Interactive Animations

Animations can interact with other DOM content by changing a groupâ€™s ID attribute to include "toggle-" followed by an ID of an element on the page that should be toggled when that group is clicked. Once clicked, a class of "layersnap-toggle-hide" will be added to the associated element, and a class of "layersnap-toggle-active" will be added to the SVG group. By default, all toggles will receive the hide class, so if you'd like to activate a group from the start, add "toggleactive" to that group's ID.

Fallback Images

To specify a fallback image for non-svg supporting browsers, give the div a background image (this rule will be negated if svg is supported).


	<div class="layersnap" data-layersnap-init style="background-image: url(path/to/fallback.png);">
	  <svg>
	    <g id="fade">...</g>
	  </svg>
	</div>

Building Animations in Adobe Illustrator

If you're editing an SVG in Adobe Illustrator, you can group graphics in the Layers palette and name their layer as if you are setting the group's ID attribute (with the transition name, duration, and delay. When you save your SVG, these layers will become `g` elements with ID attributes matching the layer names you created.

Tips and Caveats

  • In order to use the layer naming technique above, you'll need at least 2 layers in the pallete. Unnamed layers will go ignored by the Animation script.
  • If more than one layer has the same name, Illustrator will add underscores and other characters to the SVG markup to make its ID unique. It'll still appear the same way in Illustrator however, and the animation engine knows to look for these characters and ignore them.

jQuery Integration

Layersnap does not depend on jQuery, but we've created some handy tie-ins to use it with jQuery if you prefer.

To use Layersnap as a jQuery plugin and bind a few helpful events, include jQuery and the layersnap.jquery.init.js file. This file includes functions to create a $.fn.layersnap() method you can use on any jQuery object, plus event listeners to cue up layersnap animations when the enhance and inviewport events fire on layersnap elements.

To have an animation play when the page loads, give the div wrapper the attribute data-layersnap-init, and trigger an "enhance" event when the DOM is ready.


	<div class="layersnap" data-layersnap-init>
		<svg>
			<g id="fade">...</g>
		</svg>
	</div>
	

	<script>
	jQuery( function(){
		jQuery( document ).trigger( "enhance" );
	});
	</script>

Delaying Start Time for when the graphic is visible (with Porthole)

If you include Porthole in your page, you can have an animation start when it scrolls into the viewport. All of the animations in this page use Porthole to do just that.

With porthole included, to have the animation play once when it scrolls into the viewport, give the layersnap div wrapper a data-scroll-activate attribute.


	<div class="layersnap" data-scroll-activate>
	  <svg>
	    <g id="fade">...</g>
	  </svg>
	</div>

To make it replay every time it enters the viewport, also add the attribute data-scroll-deactivate.


	<div class="layersnap " data-scroll-activate data-scroll-deactivate>
	  <svg>
	    <g id="fade">...</g>
	  </svg>
	</div>
